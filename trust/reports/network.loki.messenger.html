Session

Session est un système de messagerie instantanée open-source crée par Michael Rogers, il est disponible sur Android/IOS/Ordinateur et utilise un systeme en onion pour anonymiser les connections.

Exodus Privacy
3

D'après Exodus Privacy, l'application Session contient 1 tracker de Google (Google Firebase Analytics)! Cependant, ce tracker n'est pas implémenté dans les build publiques et sera supprimé dans les prochaines releases !

Politique de confidentialité
3

Session promet ne revendre aucune donnée, tout les messages passent par un reseau en oignon pour protéger l'anonymat des utilisateurs

Chiffrement utilisé
3

Session chiffre de bout en bout les messages avec un protocole reconnu !

Open-Source
3

L'app Session est entièrement Open-Source !
